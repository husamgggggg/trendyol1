# دليل نشر موقع ترنديول على السيرفر

## معلومات السيرفر والدومين
- **الدومين**: trendyol-invest.com
- **السيرفر**: root@187.124.28.173
- **المسار على السيرفر**: `/var/www/trendyol-invest.com`

---

## الخطوات المطلوبة

### 1. إعداد DNS للدومين

قبل البدء، تأكد من إعداد سجلات DNS للدومين:

```
A Record: trendyol-invest.com → 187.124.28.173
A Record: www.trendyol-invest.com → 187.124.28.173
```

انتظر حتى يتم نشر DNS (قد يستغرق بضع دقائق إلى ساعات).

---

### 2. إعداد السيرفر (المرة الأولى فقط)

#### أ) الاتصال بالسيرفر
```bash
ssh root@187.124.28.173
```

#### ب) نسخ ملف الإعداد إلى السيرفر
من جهازك المحلي:
```bash
scp setup-server.sh root@187.124.28.173:/root/
```

#### ج) تشغيل سكريبت الإعداد على السيرفر
```bash
ssh root@187.124.28.173
chmod +x /root/setup-server.sh
/root/setup-server.sh
```

هذا السكريبت سيقوم بـ:
- ✅ تحديث النظام
- ✅ تثبيت Nginx
- ✅ تثبيت Certbot (للحصول على شهادة SSL)
- ✅ إنشاء مجلد الموقع
- ✅ إعداد Nginx
- ✅ الحصول على شهادة SSL تلقائياً
- ✅ إعداد Firewall

**ملاحظة مهمة**: عند تشغيل Certbot، سيطلب منك إدخال بريد إلكتروني. استخدم بريدك الحقيقي.

---

### 3. نشر الموقع

#### أ) إعطاء صلاحيات التنفيذ للسكريبت
```bash
chmod +x deploy.sh
```

#### ب) تشغيل سكريبت النشر
```bash
./deploy.sh
```

هذا السكريبت سيقوم بـ:
- ✅ نسخ الملفات إلى السيرفر
- ✅ إعداد Nginx
- ✅ إعادة تحميل Nginx

---

### 4. التحقق من النشر

#### أ) التحقق من الموقع
افتح المتصفح وانتقل إلى:
- http://trendyol-invest.com
- https://trendyol-invest.com (بعد تفعيل SSL)

#### ب) التحقق من SSL
```bash
curl -I https://trendyol-invest.com
```

يجب أن ترى `HTTP/2 200` أو `HTTP/1.1 200 OK`.
