# تعليمات تنظيف ونشر الموقع

## 🧹 تنظيف السيرفر فقط

إذا أردت فقط حذف الملفات القديمة من السيرفر:

```bash
./clean-server.sh
```

**ملاحظة**: سيطلب منك تأكيد العملية (انتظار 5 ثواني أو Ctrl+C للإلغاء)

أو للتنظيف التلقائي بدون تأكيد:

```bash
./clean-server-force.sh
```

---

## 🚀 العملية الكاملة (تنظيف + إعداد + نشر)

لتنفيذ جميع الخطوات دفعة واحدة:

```bash
./clean-and-deploy.sh
```

هذا السكريبت سيقوم بـ:
1. ✅ تنظيف السيرفر (حذف الملفات القديمة)
2. ✅ إعداد السيرفر (تثبيت Nginx, SSL, إلخ)
3. ✅ نشر الموقع

---

## 📋 الخطوات اليدوية

إذا أردت تنفيذ الخطوات يدوياً:

### 1. تنظيف السيرفر
```bash
./clean-server.sh
```

### 2. إعداد السيرفر (مرة واحدة فقط)
```bash
scp setup-server.sh root@187.124.28.173:/root/
ssh root@187.124.28.173
chmod +x /root/setup-server.sh
/root/setup-server.sh
```

### 3. نشر الموقع
```bash
./deploy.sh
```

---

## ⚠️ تحذيرات مهمة

- **احتفظ بنسخة احتياطية** من الملفات المهمة قبل التنظيف
- تأكد من **إعداد DNS** قبل البدء
- عند تشغيل `setup-server.sh`، سيطلب Certbot بريد إلكتروني
