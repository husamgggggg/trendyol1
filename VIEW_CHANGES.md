# كيفية رؤية التعديلات

## 🚀 الطريقة السريعة

### 1. افتح الملف مباشرة في المتصفح

**على Windows:**
```cmd
# انقر نقراً مزدوجاً على الملف
index-50.html
```

أو:
```cmd
# من سطر الأوامر
start index-50.html
```

**على Linux/Mac:**
```bash
# من سطر الأوامر
xdg-open index-50.html
# أو
open index-50.html
```

## 🌐 الطريقة الموصى بها (باستخدام خادم محلي)

### باستخدام Python:

#### Python 3:
```bash
# افتح Terminal/Command Prompt في مجلد المشروع
cd "C:\Users\it 021\Desktop\trendyol"

# شغّل الخادم
python -m http.server 8000
```

#### Python 2:
```bash
python -m SimpleHTTPServer 8000
```

**ثم افتح المتصفح على:**
```
http://localhost:8000/index-50.html
```

---

### باستخدام Node.js (http-server):

```bash
# تثبيت http-server (مرة واحدة فقط)
npm install -g http-server

# شغّل الخادم
http-server -p 8000

# ثم افتح
http://localhost:8000/index-50.html
```

---

### باستخدام PHP:

```bash
php -S localhost:8000

# ثم افتح
http://localhost:8000/index-50.html
```

## 🔍 التعديلات التي ستجدها

### 1. نظام حفظ الجلسة
- ✅ سجّل دخولك
- ✅ حدّث الصفحة (F5)
- ✅ يجب أن تبقى مسجلاً

### 2. زر الخروج الأحمر
- ✅ اضغط على "خروج"
- ✅ يجب أن يكون الزر أحمر اللون

### 3. نظام الدعم عبر Telegram
- ✅ اضغط على "💬 الدعم"
- ✅ يجب أن ترى نافذة Telegram بدلاً من البريد
- ✅ زر "📱 للتواصل اضغط هنا"

### 4. نظام الإشعارات (للمدير)
- ✅ ادخل إلى لوحة التحكم
- ✅ اضغط على "🔔 الإشعارات"
- ✅ يجب أن ترى واجهة إرسال الإشعارات

## 📝 خطوات الاختبار الكاملة

### 1. اختبار حفظ الجلسة:
```
1. افتح الموقع
2. سجّل دخولك
3. اضغط F5 (تحديث الصفحة)
4. يجب أن تبقى مسجلاً ✅
```

### 2. اختبار زر الخروج:
```
1. سجّل دخولك
2. ابحث عن زر "خروج"
3. يجب أن يكون أحمر اللون ✅
```

### 3. اختبار نظام الدعم:
```
1. اضغط على "💬 الدعم"
2. يجب أن ترى نافذة Telegram
3. يجب أن ترى زر "📱 للتواصل اضغط هنا" ✅
```

### 4. اختبار نظام الإشعارات:
```
1. ادخل كمدير (index-50.html?admin=trendyol-secret)
2. كلمة المرور: admin123
3. اضغط على "🔔 الإشعارات"
4. يجب أن ترى واجهة إرسال الإشعارات ✅
```

## ⚠️ ملاحظات مهمة

### 1. قاعدة البيانات (Supabase):
- **يجب أن تكون متصلاً بـ Supabase** ليعمل الموقع
- تأكد من تحديث URL و Key في الملف

### 2. الجداول المطلوبة:
- ✅ `users` - موجود
- ✅ `settings` - موجود
- ✅ `notifications` - **يجب إنشاؤه** (راجع NOTIFICATIONS_SETUP.md)
- ✅ `admins` - موجود

### 3. تحديث الصفحة:
- بعد أي تعديل في الكود، **حدّث الصفحة** (F5 أو Ctrl+R)
- أو اضغط Ctrl+Shift+R لإعادة تحميل كاملة (Clear Cache)

---

## 🔄 إذا لم تظهر التعديلات

### 1. تأكد من حفظ الملف:
- ✅ الملف `index-50.html` محفوظ
- ✅ لا توجد أخطاء في الكود

### 2. امسح Cache المتصفح:
```
Chrome/Edge: Ctrl+Shift+Delete → Clear Cache
Firefox: Ctrl+Shift+Delete → Clear Cache
```

أو:
```
Ctrl+Shift+R (إعادة تحميل كاملة)
```

### 3. افتح Developer Tools:
```
F12 → Console
ابحث عن أي أخطاء
```

### 4. تأكد من الاتصال بـ Supabase:
- افتح Console (F12)
- تحقق من عدم وجود أخطاء في الاتصال
