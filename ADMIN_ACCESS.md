# 🔐 كيفية الدخول إلى لوحة الأدمن

## الطريقة 1: عبر الرابط المباشر (الأسهل)

افتح المتصفح وانتقل إلى:

```
https://trendyol-invest.com?admin=trendyol-secret
```

أو:

```
http://trendyol-invest.com?admin=trendyol-secret
```

سيتم فتح صفحة تسجيل دخول الأدمن تلقائياً.

---

## الطريقة 2: عبر Console المتصفح

1. افتح الموقع: https://trendyol-invest.com
2. اضغط `F12` لفتح Developer Tools
3. اذهب إلى Console
4. اكتب:
   ```javascript
   go('adminlogin')
   ```
5. اضغط Enter

---

## تسجيل الدخول

بعد فتح صفحة الأدمن:

1. أدخل **كلمة مرور الأدمن**
2. اضغط "دخول"

**ملاحظة**: كلمة مرور الأدمن محفوظة في قاعدة البيانات Supabase في جدول `admins`.

---

## إنشاء حساب أدمن (إذا لم يكن موجوداً)

إذا لم يكن لديك حساب أدمن، يجب إنشاؤه في Supabase:

1. اذهب إلى Supabase Dashboard
2. افتح جدول `admins`
3. أضف سجل جديد:
   - `username`: `admin`
   - `password`: كلمة المرور التي تريدها

---

## تغيير كلمة مرور الأدمن

يمكن تغيير كلمة مرور الأدمن من لوحة الأدمن نفسها:

1. ادخل إلى لوحة الأدمن
2. اذهب إلى "⚙️ الإعدادات"
3. أدخل كلمة المرور الجديدة في حقل "كلمة مرور جديدة للأدمن"
4. اضغط "💾 حفظ الإعدادات"

---

## ملاحظات أمنية

- ⚠️ **لا تشارك رابط الأدمن** مع أي شخص
- ⚠️ استخدم **كلمة مرور قوية** للأدمن
- ⚠️ الرابط `?admin=trendyol-secret` يمكن تغييره في الكود لزيادة الأمان

---

## تغيير رابط الأدمن (اختياري)

لزيادة الأمان، يمكنك تغيير `trendyol-secret` إلى شيء آخر:

في ملف `index-50.html`، ابحث عن:
```javascript
if(p.get('admin')==='trendyol-secret')
```

وغيّر `trendyol-secret` إلى كلمة سرية أخرى.

---
